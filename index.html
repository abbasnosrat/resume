Projects

TFNet

Abstract

This project is the main project of my thesis. TFNet is a few-shot identification method ,which uses CNNs to identify the system's structure and parameters from its pulse response.

Status

The code is available on github.

TFNet 2

Abstract

This project is direct sequel to TFNet. Using Multi-Task learning, I am reducing the number of networks used in TFNet. This will result in a more elegant and compact identifier and solves the scalability issues of TFNet and increases its order classification accuracy.

Status

Ongoing

NLP-BASED UNIFICATION OF SALES DATABASE

Ronak Pharmaceutical Corp

Abstract

Since different distribution centers use different naming conventions for naming their customers, entities appear with different names in the database resulting in incorrect analysis. In this project, I used NLP techniques to clean and cluster entities with similar names for database unification.
